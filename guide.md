## 웹개발 과정 ( 프론트엔드 + 백엔드 )

# 프로젝트를 위해 필요한 선택지

```
1. 서버선택
2. 백엔드 언어선택
3. 데이터베이스 선택
4. 통신 방법, 아키텍트 만들기
5. 프론트엔드 ( SPA , 링크방식 )

```

# 웹개발 기술사항
```
- 백엔드 (서버)

1. OS 선택 ( window server, Linux(추천), )
    - window
    - centos (취업추천)
    - ubuntu (추천)
2. 웹서버 ( 웹언어를 해석할수 있게 도와주는 역활이기도 하고, 서버의 역활 )
    - appch (취업추천)
    - nginx (떠오르는 샛별)
    - node (자바스크립트) - socket io
3. 백엔드 언어
    - php
    - python (추천)
    - java & jsp (취업추천)
    - node
    - go lang
4. 데이터베이스
    - oracle DB
    - mysql & mariadb (취업추천)
    - mongodb (NoSQL)
5. 개발방식
    - restful api (취업추천)
    - MVC ( model - controller - view )
6. 프론트엔드
    - HTML, CSS, JS
7. 프론트방식
    - SPA ( react(배울거) , angular, vue )
    - 페이지이동 방식
8. ui/ux
    - 반응형
    - 하이브리드 ( 웹 + 모바일웹 )
    - 네이트브 ( 웹 + 앱 )
    

```
